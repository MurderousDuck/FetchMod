# FetchMod
Home of the Fetch keyword mod for SlayTheSpire  
Simple mod that adds a new keyword for the game.  
Fetch is a keyword that represents choosing a card from one of your piles and placing it into your hand.  
  
Patched Cards:  
  
All for One  
Hologram  
Secret Technique  
Secret Weapon  
Seek  
Exhume
